📍 CHAPTER 4: POINTERS – MASTER MEMORY

Pointer = variable that stores memory address. & = address-of, * = dereference (value at address).

int x = 42; int *ptr = &x; // ptr points to x printf("%d\n", *ptr); // prints 42 *ptr = 100; printf("%d", x); // x is now 100

🎯 Why pointers? Modify variables inside functions, dynamic memory, strings, arrays.

void swap(int *a, int *b) { int temp = *a; *a = *b; *b = temp; } // in main: swap(&x, &y);
✨ Pointer arithmetic: ptr+1 moves to next integer (4 bytes). Works with arrays!
